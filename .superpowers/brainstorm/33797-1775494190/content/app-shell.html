Navigation Logic

Primary — Daily Use

  • Command Center — Home. AI alerts, population KPIs, action queue
  • Patients — Registry, search, patient detail with AI summary
  • Communications — AI outreach agent, message threads, templates
  • Outcomes — Clinical metrics, ROI, HEDIS, re-cohortisation logs

Configuration — Setup & Tuning

  • Cohortisation — Tier definitions, CRS weights, scoring rules, overrides
  • Pathway Builder — Create/edit pathways (voice + drag-drop + config)

These are accessed less frequently — by program architects, not daily care managers. Separated below a divider to signal this.

Key Drill-Down Flows

Command Center alert → Patient detail → AI summary → Send outreach → Back to command center
Command Center KPI card → Outcomes deep-dive (filtered to that metric)
Patient detail → View assigned pathway → See care protocol timeline
Outcomes re-cohortisation flag → Patient detail → Approve/override tier change
