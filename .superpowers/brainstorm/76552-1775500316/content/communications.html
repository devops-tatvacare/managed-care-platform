AI Agent Behaviour

  • Autonomous mode: AI agent sends outreach based on pathway triggers (lab reminders, refill nudges, appointment scheduling). Messages are auto-generated in patient's preferred language using templates + LLM personalisation.
  • Escalation rules: No response after N attempts (configurable per pathway) → escalate to care manager. Patient replies with complexity (questions, complaints, symptoms) → route to care manager. PHQ-9 or safety keywords → immediate RN flag.
  • Care manager takeover: When escalated, AI agent pauses. Care manager sees full thread context + patient data panel. Can compose manually, use templates, or hit "AI Rewrite" to get an LLM-improved version of their draft.
  • Template library: Pre-populated chips map to pathway action types. Each template has variables (patient name, lab date, provider name) auto-filled from patient record.
  • Audit trail: Every message tagged as Auto (AI-initiated), Template (care manager used template), or Manual. Visible in patient timeline.
